Compound Odds Calculator

How it Works

1. Go to any other Calculator and Calculate the odds for whatever you need to do.

2. After getting the Odds, select the Export Button for which odds you want to record and a string will appear, copy that and paste it into the input box below.

3. Press Import and the number should appear.

4. Repeat this for however many odds you want to import to do your Compound Calculation.

5. Once finished, press Calculate and the complete odds of all the events will be Multiplied.
